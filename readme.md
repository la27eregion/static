## Méthodologie

```bash
wget \
 --recursive \
 --no-clobber \
 --page-requisites \
 --html-extension \
 --convert-links \
 --restrict-file-names=windows \
 --domains nouveauxaccords.la27eregion.fr \
 --no-parent \
 nouveauxaccords.la27eregion.fr
```
